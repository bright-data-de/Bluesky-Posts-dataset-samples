## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Die direkte URL zum jeweiligen Beitrag auf der Bluesky-Plattform. |
| `post_id` | Eine eindeutige Kennung für jeden Beitrag. |
| `post_date` | Datum und Uhrzeit, zu der der Beitrag veröffentlicht wurde. |
| `posted_by` | Informationen über den Benutzer, der den Beitrag erstellt hat, einschließlich Handle und Profil-Link. |
| `post_text` | Der Textinhalt des Beitrags. |
| `comments` | Die Anzahl der Kommentare, die zum Beitrag verfasst wurden. |
| `reposts` | Die Anzahl der Male, die der Beitrag von anderen Benutzern erneut gepostet wurde. |
| `likes` | Die Gesamtzahl der Likes, die der Beitrag erhalten hat. |
| `reposted` | Gibt an, ob der Beitrag ein Repost von Inhalten eines anderen Benutzers ist. |
| `external_links` | Links zu externen Websites, die im Beitrag enthalten sind. |
| `images` | Anzahl der an den Beitrag angehängten Bilder. |
| `videos` | Anzahl der an den Beitrag angehängten Videos. |
| `original_post` | Gibt an, ob der Inhalt original ist oder erneut gepostet wurde. |
| `pinned` | Gibt an, ob der Beitrag oben im Profil des Benutzers angeheftet ist. |
| `discovery_input` | Keywords oder Tags, die zur Inhaltsentdeckung verwendet werden. |
| `contains_video` | Gibt an, ob der Beitrag ein Video enthält. |
| `profile_followers_count` | Die Anzahl der Follower, die der Ersteller des Beitrags hat. |
| `profile_follows_count` | Die Anzahl der Benutzer, denen der Ersteller des Beitrags folgt. |
| `profile_posts_count` | Gesamtzahl der vom Benutzer veröffentlichten Beiträge. |
| `post_type` | Der Beitragstyp, z. B. Text, Bild oder Video. |
| `video_thumbnails` | Vorschaubilder (Thumbnails) für alle im Beitrag enthaltenen Videos. |
| `video_duration` | Dauer aller Videos im Beitrag, in Sekunden. |


## Use Cases

### 1. Social Media Sentiment Analysis
Analysieren Sie die öffentliche Stimmung zu verschiedenen Themen, indem Sie den Textinhalt und die Engagement-Kennzahlen von Beiträgen untersuchen.

### 2. Influencer Impact Assessment
Bewerten Sie den Einfluss verschiedener Benutzer, indem Sie Follower-Zahlen und Engagement-Ratios analysieren, um wichtige Influencer zu identifizieren.

### 3. Content Performance Benchmarking
Vergleichen Sie die Performance von Beiträgen über verschiedene Profile hinweg, um Best Practices zu ermitteln und Content-Strategien zu optimieren.
