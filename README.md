## Datenpunkte

| Feldname | Beschreibung |
|------------|-------------|
| `url` | Die direkte URL zum spezifischen Beitrag auf der Bluesky-Plattform. |
| `post_id` | Eine eindeutige Kennung für jeden Beitrag. |
| `post_date` | Das Datum und die Uhrzeit, zu der der Beitrag veröffentlicht wurde. |
| `posted_by` | Informationen über den Benutzer, der den Beitrag erstellt hat, einschließlich Handle und Profil-Link. |
| `post_text` | Der Textinhalt des Beitrags. |
| `comments` | Die Anzahl der Kommentare zu dem Beitrag. |
| `reposts` | Die Anzahl der Male, die der Beitrag von anderen Benutzern erneut gepostet wurde. |
| `likes` | Die Gesamtzahl der Likes, die der Beitrag erhalten hat. |
| `reposted` | Gibt an, ob der Beitrag ein Repost des Inhalts eines anderen Benutzers ist. |
| `external_links` | Links zu externen Websites, die im Beitrag enthalten sind. |
| `images` | Anzahl der an den Beitrag angehängten Bilder. |
| `videos` | Anzahl der an den Beitrag angehängten Videos. |
| `original_post` | Gibt an, ob der Inhalt original oder erneut gepostet ist. |
| `pinned` | Gibt an, ob der Beitrag oben im Profil des Benutzers angeheftet ist. |
| `discovery_input` | Schlüsselwörter oder Tags, die zur Inhaltsfindung verwendet werden. |
| `contains_video` | Gibt an, ob der Beitrag ein Video enthält. |
| `profile_followers_count` | Die Anzahl der Follower, die der Ersteller des Beitrags hat. |
| `profile_follows_count` | Die Anzahl der Benutzer, denen der Ersteller des Beitrags folgt. |
| `profile_posts_count` | Gesamtzahl der vom Benutzer erstellten Beiträge. |
| `post_type` | Der Typ des Beitrags, z. B. Text, Bild oder Video. |
| `video_thumbnails` | Vorschaubilder für alle im Beitrag enthaltenen Videos. |
| `video_duration` | Länge aller Videos im Beitrag in Sekunden. |


## Anwendungsfälle

### 1. Sentiment-Analyse in sozialen Medien
Analysieren Sie die öffentliche Stimmung zu verschiedenen Themen, indem Sie den Textinhalt und die Engagement-Metriken von Beiträgen untersuchen.

### 2. Bewertung der Influencer-Wirkung
Bewerten Sie den Einfluss verschiedener Benutzer, indem Sie Follower-Zahlen und Engagement-Raten analysieren, um wichtige Influencer zu identifizieren.

### 3. Benchmarking der Content-Performance
Vergleichen Sie die Performance von Beiträgen über verschiedene Profile hinweg, um Best Practices zu bestimmen und Content-Strategien zu optimieren.
